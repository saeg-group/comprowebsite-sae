# Boilerplate Website Company Profile - SAE Aqiqah

Ini adalah struktur folder & file dasar untuk pembuatan halaman *Company Profile* satu halaman statis yang modern milik **SAE Aqiqah**.

## Struktur Folder
```
comprowebsite-sae/
├── index.html          # File HTML utama (struktur layout kerangka halaman)
├── css/
│   └── styles.css      # File stylesheet utama (Desain visual & Variabel Merek)
├── js/
│   └── main.js        # File interaktivitas Javascript (Navigasi & Scroll)
├── assets/
│   ├── images/         # Tempat penyimpanan gambar & aset foto produk/kandang
│   └── icons/          # Tempat penyimpanan file ikon visual
└── README.md           # Petunjuk penggunaan boilerplate ini
```

## Cara Penggunaan
1. **Gaya Desain**: File `css/styles.css` sudah dideklarasikan dengan variabel palet warna Tanamera Aesthetic (`--onyx-black`, `--crimson-red`, `--gold`, dsb). Anda tinggal memetakan visual layout sesuai dengan detail di [Rancangan_Company_Profile.md](../Rancangan_Company_Profile.md).
2. **Pengisian Konten**: Kerangka di `index.html` sudah siap diisi teks informatif dan humanis yang selaras dengan nilai-nilai brand SAE Aqiqah (Syar'i, Amanah, Excellent).
3. **Pengisian Aset**: Tempatkan file logo asli (`logo.svg` atau `logo.png`) serta foto hasil photoshoot kandang/produk di folder `assets/images/`.
